Aturan & Format Parameter Dataset

1. Data Mahasiswa (mahasiswa.csv / .xlsx)

Kolom (Parameter) Tipe Data Keterangan / Aturan Pengisian
NIM Teks / Angka Nomor Induk Mahasiswa unik (Kunci Utama). Tidak boleh kosong atau duplikat.
Nama Teks Nama lengkap mahasiswa.

2. Struktur Kuis (kuis_struktur.csv / .xlsx)

Kolom (Parameter) Tipe Data Keterangan / Aturan Pengisian
ID_Soal Teks ID unik untuk setiap soal kuis (Kunci Utama, contoh: Q001).
Materi Teks Kategori/Topik materi utama kuis (contoh: Manajemen Proses). Wajib ditulis konsisten.
Submateri Teks Sub-topik materi (contoh: Sinkronisasi).
Bobot Angka (Int) Tingkat kesulitan soal (umumnya bernilai 1, 2, atau 3). Hanya opsional (tidak masuk kalkulasi AI).
Opsi_A s.d. Opsi_D Teks Pilihan jawaban kuis.
Kunci Huruf Kunci jawaban benar. Wajib bernilai satu huruf kapital: A, B, C, atau D.

3. Log Jawaban Mahasiswa (mahasiswa_log_jawaban.csv / .xlsx)

Kolom (Parameter) Tipe Data Keterangan / Aturan Pengisian
NIM Teks / Angka NIM mahasiswa yang menjawab. Wajib terdaftar di Data Mahasiswa.
ID_Soal Teks ID Soal kuis yang dijawab. Wajib terdaftar di Struktur Kuis.
Jawaban_Mahasiswa Huruf Pilihan yang dijawab (A, B, C, atau D).
Skor_Biner Angka (Biner) Hasil koreksi jawaban. Wajib bernilai 0 (jawaban salah) atau 1 (jawaban benar). Digunakan untuk perhitungan probabilitas AI BKT.
